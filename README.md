
# Basic 2 Screen App

I created a basic 2 screen app that include a static data with tableview. Click on the tableview and redirecting the second screen according to tapped index.


## Used

**Swift** ✅

**TableView** ✅

**ImageView** ✅

**Segue** ✅

**Data Transfer** ✅
